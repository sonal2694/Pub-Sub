Background:
-----------
The folder contains the following files:
1. index.html: client of the application
2. app.js: server of the application
3. Dockerfile: contains docker instructions that runs the application


Instructions on how to run Phase 1:
-----------------------------------

1.On the terminal, run the following docker commands:
    - docker build -t sonal/phase1
    - docker run -p 3000:3000 sonal/phase1

3. Open the "index.html" file on a web browser.
4. Type in a simple Python program in the textbox and click on "Execute".
5. The output of the program is displayed on the same page.
